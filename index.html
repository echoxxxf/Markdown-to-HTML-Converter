Markdown → HTML コンバータ

Markdownを貼り付けて「変換＆ダウンロード」を押すと、見出し毎にアコーディオン構造にまとめたHTMLを生成します。

対応済Markdown記法
  1. 見出し
    • h1〜h6タグに対応済。
    • 例:
      # Heading 1
      ## Heading 2
  2. 改行・空行
    • 改行のみで対応。行末のbrタグや半角スペースは改行には影響せず、そのまま出力される
    • 1行空けも2行空けも得られる結果は同様。pタグで対応されるため、空行をたくさん入れても空白スペースは増えない。
  3. リスト
    • 例:
      - Unordered list item 1
        - Nested item 1
        - Nested item 2
      - Unordered list item 2
  4. 番号付きリスト
    • 例:
      1. Ordered item 1
      2. Ordered item 2
        1. Nested ordered 1
        2. Nested ordered 2
  5. コード
    • インライン:
      `テキストやコード`
    • 複数行:
      ```
      コード1
      コード2
      コード3
      ```
  6. 引用
    • 例:
      >This is a blockquote.
      >It can span multiple lines and include **Markdown** syntax.
  7. リンク
    • 例:
      [タイトル](URL)
  8. 画像
    • 例:
      ![代替テキスト](URL)
  9. テーブル
    • 例: 
      | Name       | Age | City      |
      |------------|-----|-----------|
      | Alice      | 25  | New York  |
      | Bob        | 30  | London    |
      | Charlie    | 28  | Tokyo     |
  10. 太字
    • 例:
      **bold**
  11. 斜体
    • 例:
      *italic*
  12. 下線
    • 例:
      __underline__
  13. 打ち消し線
    • 例:
      ~~strikethrough~~
  14. 水平線
    • 例:
      ---
      or
      ***
      or
      ___
  15. GitHub Alerts
    • 例:
      >[!NOTE]
      >Useful information that users should know, even when skimming content.

      >[!TIP]
      >Helpful advice for doing things better or more easily.

      >[!IMPORTANT]
      >Key information users need to know to achieve their goal.

      >[!WARNING]
      >Urgent info that needs immediate user attention to avoid problems.

      >[!CAUTION]
      >Advises about risks or negative outcomes of certain actions.

特殊な文字列の装飾
  1. 文字列の置換
    • {}で囲った文字列は、ファイルを開く度にテキストの置換が可能です。
    • 例: {HO1} → 開いたときに「花子」などに置換可能
  2. フォント色
    • デフォルト（赤色）
      • @@テキスト@@で囲った文字列は、文字色が赤色（#FF0000）になります。
      • 例: @@HO1@@ → HO1と表示
    • 色指定
      • @@#RRGGBB テキスト@@で囲った文字列は、指定した色で文字が表示されます（3桁・6桁カラーコード対応）。
      • 例: @@#f00 赤文字@@ → 赤文字と表示
      • 例: @@#0054ff 青文字@@ → 青文字と表示
  3. ハイライト
    • デフォルト（橙色）
      • %%テキスト%%で囲った文字列は、文字背景が橙色（#FFA500）になります。
      • 例: %%Emphasis%% → Emphasisと表示
    • 色指定
      • %%#RRGGBB テキスト%% で囲った文字列は、指定した色で文字の背景色を変更できます（3桁・6桁カラーコード対応）。
      • 例: %%#a6ff4a 背景色%% → 背景色と表示
  4. # ᴄʜᴇᴄᴋʟɪꜱᴛ 以降の内容
    • この見出し以降の内容を、画面右下のボトムシートに設定できます。チェックリストとして活用してください。
    • この記述は、すべての内容を入力し終えた最後に入れてください。
    • 例:
      # ᴄʜᴇᴄᴋʟɪꜱᴛ
      - [ ] xxx
      - [ ] xxx
